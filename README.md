# FUTURE_CS_03
API Security Risk Analysis of JSONPlaceholder API using Postman.

## Objective
Perform a read-only security assessment of a public API.

## Target API
JSONPlaceholder API

## Tools Used
- Postman
- Browser Developer Tools
- GitHub

## Methodology
- Documentation Review
- Endpoint Testing
- Header Analysis
- Response Analysis
- Risk Assessment

## Findings
1. Unauthenticated Access
2. Excessive Data Exposure
3. Weak Access Control
4. Missing Authentication Tokens
5. Missing Rate Limiting

## Outcome
Created a professional API Security Risk Analysis Report with remediation recommendations.
